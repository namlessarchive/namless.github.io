How to Install and Play Nameless (2005 UPDATE)

Written by CotillardMain78

This guide will breifly explain in steps on how to download and install the Namesless game with ease. Has been working as of 2005.

Step 1.

Secure a PSX Emulator. There are variety of programs that work for this, aswell as mulit-console emulators. Most Playstation 1 Emulators work. Here are some of our favorites.

  • PCSX (Ver 1.5)
  • DuckStation (BIOS Required)
  • PCsx Reloaded
  • Retroarch (BIOS Required)

Step 2.

If you're using an emulator that requires a BIOS for the PSX. Then you will have to aquire the Bios file from your own PSX, legally of course. Once this file has been ripped, you can load it into your emulator. If you do not have a bios file, you will have to find a method to rip it. (of course, we would never provide such files on our site.)

Step 3.

Next, you will need the Nameless Rom File. Aquiring it can be a bit hard, but try to stick with us. First, download the main file here.

Main File

Now you will need the first patch file, here is a direct tutorial on how to apply the patch if you don't know already

Patch+Patching guide

Step 4.

Once you have the main file and the patch applied, you'll need to begin the loading process. Make sure you have Nameless on the root of your drive, your emulator location doesn't matter, however if you have problems, be sure the program root is also the same.
To load, simply drag the patch file and the main file together in the emulator. Then use the decrypter to underline the two files.

Step 5.

Once loading has finished, you can open and play Nameless! Congrats and enjoy the game. You can use our other guides to help you along, if any of the downloads here are not working feel free to contact the forum mods.

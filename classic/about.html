About Nameless!

Nameless was a 1997 audio/visual interative program (Video Game) that released on the Playstation PSX. It features a Semi-Open World with heavy JPRG elements.


The Game

The overworld was a simple mapcrawl much like ealier JPRGs of the era. It connects to more detailed areas like towns, dungeons, interiors, etc, transitioning to Parasite Eve (and by proxy, classic Resident Evil) type exploration. Running into encounters in overworld would start simple turn based combat, with less detailed sprites of party members, however in more detailed areas, there is a time based combat mode. The game had a lot of variety, especially with more “far out” sections starting to have more LSD Dream Emulator type exploration rather then the RPG the game is supposed to be. Regardless the game was only released in ███████, 1997. Fans have only been able to find fringe copies of the game in later years, as the studio behind Nameless went bankrupt shortly after release, all archives of the game were lost apparently. The copies found by fans were then documented on RPG forums during early 2000s. However, the forum moderators were not convinced that Nameless was a real game and discussion of it was often punished. Alas, the domain of the original Nameless website was in limbo. The site had all of it’s information redacted but the domain was transferred to an upcoming Angelfire clone hosting service. Nameless fans decided to take the domain and recover what they could of the site, hence, the Nameless Offical Fan Project was born.
